# Гипотезы для тестирования промптов

Формулировки и структурные решения, которые нужно проверить через autoresearch-цикл. Каждая гипотеза — изменение в одном промпте, измеряется через `score = coverage × consistency`.

| # | формулировка | гипотеза |
|---|---|---|
| H1 | «Ты — нарратолог, ревизор сюжетной структуры» | Базовая |
| H2 | «Ты — нарратолог, специалист по сериальной драматургии» | Акцент на экспертизу, а не на проверку — может дать более глубокий анализ |
| H3 | «Ты — редактор сценарного отдела» | Индустриальная роль — может быть практичнее теоретика |
| H4 | «Ты — шоураннер, проверяющий работу райтерз-рума» | Максимальный авторитет — может давать более решительные вердикты |
| H5 | Без указания роли, только задача | Контрольная группа — роль вообще не нужна? |

| # | вариант | гипотеза |

## Как тестировать

Одна гипотеза = один эксперимент в autoresearch-цикле:
1. Изменить одну переменную в промпте
2. Прогнать N раз (минимум 3)
3. Вычислить score = coverage × consistency
4. Сравнить с базовым score
5. Keep / discard
